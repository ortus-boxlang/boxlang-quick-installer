## [1.0.2] - 2024-06-21

- Use the `latest` as the default now

## [1.0.2] - 2024-06-20

- Remove old zips upon first install
- Updated release build
- Java 21 as the new standard

## [1.0.1] - 2024-May-28

### Added

The installer scripts are now installed to: `/usr/local/bin`. So you can reuse them.

- `install-boxlang`
- `install-bx-module`

## [1.0.0] - 2024-May-27

### Added
